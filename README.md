# Projet Recat avec Bootstrap 
Ce projet est une application React responsive sur les différents supports
il utilise la librairie CSS Bootstrap ainsi que la librairie Javascript React

# Prerequis

aprés l'installation de Node.js
lancer l'invit de commande puis dans le dossier ou l'on souhaite créer notre projet : 
    - lancer la commande npx-create-app nom du projet : celà pour installer la librairie react dans notre projet
    -puis dans le dossier (en utilisant les commande cd pour naviguer) : lancer la commande npm start pour lancer le serveur local
    -puis lancer la commande code . pour lancer Visual studio code (VS CODE)

à partir de VS CODE lancer le terminal et installer Bootstrap (commande  npm install bootstrap)
